## Overview

VisaVerse Copilot is a clean, minimal Streamlit web application that provides:
- **Visa & Work Eligibility Assistant:** Personalized visa recommendations based on your profile
- **Document Readiness Checker:** Ensure all required documents are prepared
- **Cultural & Communication Guide:** Navigate cultural differences in your destination country

## Installation

1. Clone this repository:
```bash
git clone https://github.com/ravigohel142996/VisaVerse-Ai.git
cd VisaVerse-Ai
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the Streamlit app:
```bash
streamlit run app.py
```

The app will open in your default browser at `http://localhost:8501`

## Design Philosophy

VisaVerse Copilot follows a clean, professional SaaS design:
- Light background with centered content (max-width 900px)
- Card-based layout with rounded corners and soft shadows
- Clear headings and generous spacing
- Minimal styling focused on readability
- Professional look inspired by Notion and Stripe

## Technology Stack

- **Framework:** Streamlit
- **Language:** Python 3.8+
- **Styling:** Custom CSS (no external frameworks)
